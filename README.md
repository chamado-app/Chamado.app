# 🛸 Chamdo.app

## 📦 Prerequisites

- Node.js ^18 or ^16 or ^14.19
- Yarn 1.21.1

## 🛬 Cloning

```sh
git clone git@gitlab.com:chamado.app/chamado.app.git
# or
git clone https://gitlab.com/chamado.app/chamado.app.git
```

```sh
cd chamado.app
```

## 🏃 Running

Install node libraries

```sh
yarn install
```

Running in dev mode

```sh
yarn dev
```

Building for production

```sh
yarn build
```
